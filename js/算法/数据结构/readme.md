# 数据结构

## 简洁

数据结构是计算机存储、组织数据的方式。简单来说就是将一组数据通过一种或者多种特定的方式组成一个集合，这种集合的结构就叫做数据结构。

## 常见的数据结构预览

1. 栈（ Stack ）
2. 队列（Queue）
3. 链表（Linked List）
4. 集合（Set）
5. 哈希表/散列表（Hash Table）
6. 树（Tree）
7. 图（Graph）
8. 字典树（Trie）
9. 堆（Heap）
